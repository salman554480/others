Script Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside yourWebsite folder.
  2. Cpanel Upload:
    • Open your hosting cpanel and navigate to File Manager
    • Browse to required directory (Normally public_html)
    • Upload the files inside yourWebsite folder.
  • Database Connectivity:
    1. Open your hosting cpanel and navigate to Database Management
    2. Create Database, Username and set the Password for username
    3. Update the database credentials in db.php file which is place in directory admin/parts/db.php
